Build AI-Civ TestUniverse Showcase Website

  Build a complete, production-ready showcase website for AI-Civ TestUniverse in website/ directory.

  Tech Stack:
  - Framework: Next.js 14 (App Router)
  - Styling: Tailwind CSS + shadcn/ui components
  - Animations: Framer Motion
  - Icons: Lucide React
  - Code highlighting: Shiki
  - Diagrams: Mermaid.js (rendered server-side)
  - Documentation: MDX
  - Deployment: Static export (works on Vercel, Netlify, GitHub Pages)

  Design System:
  - Primary color: #3B82F6 (blue-500) - trust, technology
  - Accent color: #10B981 (emerald-500) - success, confidence
  - Warning: #F59E0B (amber-500)
  - Error: #EF4444 (red-500)
  - Background: #0F172A (slate-900) for dark mode, #FFFFFF for light
  - Font: Inter for body, JetBrains Mono for code
  - Dark mode default, with toggle

      ```python
      from kernel.plugin_sdk import PluginBase, PluginMeta, PluginCapabilities
      from kernel.models import Finding, TargetType, TestFamily, Severity

      class MyToolPlugin(PluginBase):
          """Adapter for MyTool testing framework."""

          def meta(self) -> PluginMeta:
              return PluginMeta(
                  id="plugin.mytool",
                  name="MyTool",
                  version="1.0.0",
                  author="Your Name",
                  tool_name="mytool",
                  tool_version=">=2.0"
              )

          def capabilities(self) -> PluginCapabilities:
              return PluginCapabilities(
                  target_types=[TargetType.WEB],
                  families=[TestFamily.SECURITY],
                  evidence_types=["json", "html"]
              )

          def configure(self, target, policy, plan) -> ExecutionSpec:
              return ExecutionSpec(
                  image="mytool/mytool:latest",
                  command=["mytool", "scan", target.url],
                  timeout_seconds=300,
                  resource_limits=ResourceLimits(memory_mb=512)
              )

          def normalize(self, result, artifacts) -> List[Finding]:
              findings = []
              for issue in result.output["issues"]:
                  findings.append(Finding(
                      title=issue["title"],
                      severity=self._map_severity(issue["severity"]),
                      family=TestFamily.SECURITY,
                      # ... map all fields
                  ))
              return findings
      ```
